## Project
Build a personal website with 25 landing page iterations and 1 gallery page.

## Hard constraints
- The root index.html must be the gallery page.
- Version pages should live under /pages as v01.html to v25.html.
- Use pure HTML and CSS ONLY.
- No frameworks.
- No external APIs.
- No data persistence.
- Keep the project easy to deploy on Vercel.

## Process
- v01-v10 should explore very different directions.
- v11-v19 should narrow and combine promising ideas.
- v20-v25 should refine toward a final choice.

## Content
- The site is personal rather than corporate.
- It can include story, interests, projects, identity, links, or other relevant sections.
- Do not force resume-style structure unless requested.

## Working style
- Plan before coding.
- Work in small batches.
- Do not overwrite the root index.html.
- Update the gallery as new versions are added.
- Suggest commit messages, but do not auto-commit unless asked.
